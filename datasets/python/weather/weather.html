Overview

Under the National Oceanic and Atmpospheric Administration, the National Weather Service provides daily weather reports for cities across the county. This is done through the use of 122 different Weather Forcast Offices throughout the country. These WFOs are responsible for the daily weather reports for serveral cities throughout their region of coverage. This data set takes the information from these WFO reports for cities across the country through the months of April to June of 2016.

http://w2.weather.gov/climate/

Usage

This library has 3 functions you can use. 
import weather
list_of_report = weather.get_weather()
list_of_report = weather.get_weather_by_day(6-1-16)
list_of_report = weather.get_weather_by_city(Blacksburg)
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import weather
# These may be slow!
list_of_report = weather.get_weather(test=False)
list_of_report = weather.get_weather_by_day(6-1-16, test=False)
list_of_report = weather.get_weather_by_city(Blacksburg, test=False)

Documentation

weather.get_weather(test=True)

Returns weather reports from the dataset.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

weather.get_weather_by_day(day, test=True)

Given a day, returns all the weather reports for that day in the database.

  • day: Str

    The day to get reports.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

weather.get_weather_by_city(city, test=True)

Given a city, returns all the weather reports for that city in the database.

  • city: Str

    The city to get reports.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
