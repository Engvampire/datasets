Overview

Originally collected by Dr. Joe Cox, this dataset has information about the sales and playtime of over a thousand video games released between 2004 and 2010. The playtime information was collected from crowd-sourced data on "How Long to Beat".

Some more information can be found here.

Cox, Joe. "What makes a blockbuster video game? An empirical analysis of US sales data." Managerial and Decision Economics 35.3 (2014): 189-198.

Usage

This library has 1 function you can use. 
import video_games
list_of_video_games = video_games.get_games()
Additionally, some of the functions can return a sample of the Big Data using an extra argument. If you use this sampled Big Data, it may be much faster. When you are sure your code is correct, you can remove the argument to use the full dataset. 
import video_games
# These may be slow!
list_of_video_games = video_games.get_games(test=True)

Documentation

 video_games.get_games(test=False)

Returns information about all the games

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
