Overview

This dataset has financial records of New Orleans slave sales, 1856-1861. There are almost 16,000 sales recorded in this dataset.

Calomiris, Charles W., and Jonathan Pritchett. 2016. "Betting on Secession: Quantifying Political Events Surrounding Slavery and the Civil War." American Economic Review, 106(1): 1-23.

Downloads

Download all of the following files.

  1. slavery.py
  2. slavery.db

Usage

This library has 1 function you can use. 
import slavery
list_of_tracsaction_record = slavery.get_transaction()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import slavery
# These may be slow!
list_of_tracsaction_record = slavery.get_transaction(test=False)

Documentation

slavery.get_transaction(test=True)

Returns a list of the transactions in the database.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
