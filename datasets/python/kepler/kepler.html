Overview

The NASA Exoplanet Archive is an online astronomical exoplanet and stellar catalog and data service that collates and cross-correlates astronomical data and information on exoplanets and their host stars and provides tools to work with these data. The Exoplanet Archive is dedicated to collecting and serving important public data sets involved in the search for and characterization of extrasolar planets and their host stars. The data include stellar parameters (such as positions, magnitudes, and temperatures), exoplanet parameters (such as masses and orbital parameters) and discovery/characterization data (such as published radial velocity curves, photometric light curves, images, and spectra).

Usage

This library has 5 functions you can use. 
import kepler
list_of_exoplanet = kepler.get_exoplanets()
a_record = kepler.get_name_by_id("11446443")
list_of_koi = kepler.get_objects_of_interest()
list_of_stellar = kepler.get_stellar()
list_of_tce = kepler.get_threshold_crossing_events()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import kepler
# These may be slow!
list_of_exoplanet = kepler.get_exoplanets(test=False)
list_of_koi = kepler.get_objects_of_interest(test=False)
list_of_stellar = kepler.get_stellar(test=False)
list_of_tce = kepler.get_threshold_crossing_events(test=False)

Documentation

kepler.get_exoplanets(test=True)

Returns information about Exoplanets discovered by Kepler. An exoplanet or extrasolar planet is a planet that orbits a star other than the Sun.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

kepler.get_name_by_id(kepid)

Returns information about the name of a given target.

  • kepid: Str

    Target identification number, as listed elsewhere in the data.

kepler.get_objects_of_interest(test=True)

Returns the Kepler Objects of Interests (KOI), targets identified by the Kepler Project that displays at least one transit-like sequence within Kepler time-series photometry that appears to be of astrophysical origin and initially consistent with a planetary transit hypothesis.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

kepler.get_stellar(test=True)

Returns stellar information about various objects of interest.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

kepler.get_threshold_crossing_events(test=True)

Returns information about Threshold Crossing Events (TCE), sequences of transit-like features in the flux time series of a given target that resembles the signature of a transiting planet to a sufficient degree that the target is passed on for further analysis.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
