Overview

This dataset is all about over 600 movies. It was collected and organized by the Cornell Movie Dialogs Corpus. More information about the project is available at the CMDC site (http://www.mpi-sws.org/~cristian/Cornell_Movie-Dialogs_Corpus.html).

Downloads

Download all of the following files.

  1. scripts.py
  2. scripts.db

Usage

This library has 2 functions you can use. 
import scripts
list_of_script = scripts.get_scripts()
a_script = scripts.get_script_by_title("\"the princess bride\"")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import scripts
# These may be slow!
list_of_script = scripts.get_scripts(test=False)

Documentation

scripts.get_scripts(test=True)

Returns all the scripts

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

scripts.get_script_by_title(title)

Returns a specific script, given it's title.

  • title: Str

    The title of the movie. There are only 600 movies in the database, but there are many popular ones.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.
