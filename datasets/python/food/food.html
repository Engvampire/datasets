Overview

The following data comes from the United States Department of Agriculture's Food Composition Database. It contains data for various types of food including the amounts of different vitamins and minerals found in the foods as well as macronutrient percentages. The food covered spans a large variety of foods from butter to Campbell's soup.

https://ndb.nal.usda.gov/

Usage

This library has 1 function you can use. 
import food
list_of_report = food.get_reports()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import food
# These may be slow!
list_of_report = food.get_reports(test=False)

Documentation

food.get_reports(test=True)

Returns food reports from the dataset.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
