Overview

Collected statistics about the tallest buildings all over the world, including the materials used to develop them, the purpose of the buildings, and more.

http://www.skyscrapercenter.com/

Usage

This library has 1 function you can use. 
import skyscrapers
list_of_skyscraper = skyscrapers.get_skyscrapers()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import skyscrapers
# These may be slow!
list_of_skyscraper = skyscrapers.get_skyscrapers(test=False)

Documentation

skyscrapers.get_skyscrapers(test=True)

Returns a list of the skyscrapers in the database.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
