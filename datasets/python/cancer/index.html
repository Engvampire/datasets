Overview

From the United States Cancer Statistics as part of the U.S. Center for Disease Control, the following data set focuses on the crude rate for all types of cancer reported for different demograpic groups. Significant groupings include age, gender, race and geographical area.

http://www.cdc.gov/cancer/npcr/uscs/download_data.htm

Usage

This library has 3 functions you can use. 
import cancer
list_of_report = cancer.get_reports()
list_of_report = cancer.get_reports_by_year("1999")
list_of_report = cancer.get_reports_by_area("Alabama")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import cancer
# These may be slow!
list_of_report = cancer.get_reports(test=False)
list_of_report = cancer.get_reports_by_year("1999", test=False)
list_of_report = cancer.get_reports_by_area("Alabama", test=False)

Documentation

cancer.get_reports(test=True)

Returns cancer reports from the dataset.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

cancer.get_reports_by_year(year, test=True)

Given a year, returns all the cancer reports for that year in the database.

  • year: int

    The year to get reports.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

cancer.get_reports_by_area(area, test=True)

Given a area, returns all the cancer reports for that area in the database.

  • area: str

    The year to get reports.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
