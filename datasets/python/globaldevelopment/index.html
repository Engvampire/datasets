Overview

The following data contains records collected on different countries and geographic locations from 1980 - 2013 from the World Bank. Included is different data about urban development, agriculture and rural development, health, and infrastructure.

https://ndb.nal.usda.gov/

Downloads

Download all of the following files.

  2. globaldevelopment.db

Usage

This library has 3 functions you can use. 
import globaldevelopment
list_of_report = globaldevelopment.get_reports()
list_of_report = globaldevelopment.get_reports_by_year("1990")
list_of_report = globaldevelopment.get_reports_by_country("'Afghanistan'")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import globaldevelopment
# These may be slow!
list_of_report = globaldevelopment.get_reports(test=False)
list_of_report = globaldevelopment.get_reports_by_year("1990", test=False)
list_of_report = globaldevelopment.get_reports_by_country("'Afghanistan'", test=False)

Documentation

globaldevelopment.get_reports(test=True)

Returns global development reports from the dataset.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

globaldevelopment.get_reports_by_year(year, test=True)

Returns global development reports for a specified year.

  • year: int

    The desired year

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

globaldevelopment.get_reports_by_country(country, test=True)

Returns global development reports for a specified country.

  • country: int

    The desired country

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
