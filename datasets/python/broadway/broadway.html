Overview

This library holds data about Broadway shows, made available by the Broadway League (the national trade association for the Broadway industry). You can view the data online at http://www.broadwayleague.com/.

Usage

This library has 2 functions you can use. 
import broadway
list_of_production = broadway.get_shows()
list_of_production = broadway.get_show_by_theatre("friedman")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import broadway
# These may be slow!
list_of_production = broadway.get_shows(test=False)
list_of_production = broadway.get_show_by_theatre("friedman", test=False)

Documentation

broadway.get_shows(test=True)

Returns information about all the shows

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

broadway.get_show_by_theatre(theatre, test=True)

Returns information about all the shows at a given theatre

  • theatre: Str

    The name of the theatre (e.g., "friedman").

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
