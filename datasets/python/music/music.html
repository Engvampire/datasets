Overview

The Music library has data about a multitude of songs and artists.

Usage

This library has 3 functions you can use. 
import music
a_song = music.get_song_by_name("I Didn't Mean To")
list_of_song = music.get_songs_by_artist('Aerosmith')
list_of_song = music.get_songs()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import music
# These may be slow!
list_of_song = music.get_songs_by_artist('Aerosmith', test=False)
list_of_song = music.get_songs(test=False)

Documentation

music.get_song_by_name(title)

Given the title of a song, returns information about the song.

  • title: str

    The title of the song.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

music.get_songs_by_artist(artist, test=True)

Given the name of an artist, returns all the songs by that artist in the database.

  • artist: str

    The name of the artist or band.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

music.get_songs(test=True)

Gets a list of all the songs in the database.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
