Usage

This library has 5 functions you can use. 
import nfl
list_of_game = nfl.get_games_by_home_team("\"St. Louis Cardinals\"")
list_of_game = nfl.get_games_by_away_team("\"St. Louis Cardinals\"")
list_of_game = nfl.get_games()
list_of_player = nfl.get_players()
a_player = nfl.get_player_by_name("'Ameer Abdullah'")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import nfl
# These may be slow!
list_of_game = nfl.get_games(test=False)
list_of_player = nfl.get_players(test=False)

Documentation

nfl.get_games_by_home_team(home_team)

Returns information about a single football game.

  • home_team: Str

    The name of the football team (e.g., "St. Louis Cardinals")

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

nfl.get_games_by_away_team(away_team)

Returns information about a single football game.

  • away_team: Str

    The name of the football team (e.g., "St. Louis Cardinals")

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

nfl.get_games(test=True)

Returns information about a single football game.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

nfl.get_players(test=True)

Returns information about all the football players.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

nfl.get_player_by_name(name)

Returns information about a single football player

  • name: Str

    The name of the player (e.g., "Ameer Abdullah")

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.
