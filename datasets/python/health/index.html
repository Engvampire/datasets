Overview

This library holds data about reports of diseases in America.

Downloads

Download all of the following files.

  1. health.py
  2. health.db

Usage

This library has 4 functions you can use. 
import health
list_of_report = health.get_reports_by_year("1990")
list_of_report = health.get_reports_by_disease("'MUMPS'")
list_of_report = health.get_reports_by_location("'Delaware'")
list_of_report = health.get_all_reports()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import health
# These may be slow!
list_of_report = health.get_reports_by_year("1990", test=False)
list_of_report = health.get_reports_by_disease("'MUMPS'", test=False)
list_of_report = health.get_reports_by_location("'Delaware'", test=False)
list_of_report = health.get_all_reports(test=False)

Documentation

health.get_reports_by_year(year, test=True)

Returns information about the status of disease for the given year, from 1928 to 2011.

  • year: int

    A year between 1928 and 2011.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

health.get_reports_by_disease(disease, test=True)

Returns information about the status of disease for the given disease.

  • disease: str

    A disease such as 'HEPATITIS A', 'MEASLES', 'MUMPS', 'PERTUSSIS', 'POLIO', 'RUBELLA', or 'SMALLPOX'.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

health.get_reports_by_location(location, test=True)

Returns information about the status of disease for the given location.

  • location: str

    A state such as "Alabama"

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

health.get_all_reports(test=True)

Returns information about the status of disease for all locations, diseases, and years.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
