Overview

This is a dataset about cars and how much fuel they use.

  2. cars.db

Usage

This library has 3 functions you can use. 
import cars
list_of_car = cars.get_cars()
list_of_car = cars.get_cars_by_year("2001")
list_of_car = cars.get_cars_by_make("'Pontiac'")
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import cars
# These may be slow!
list_of_car = cars.get_cars(test=False)
list_of_car = cars.get_cars_by_year("2001", test=False)
list_of_car = cars.get_cars_by_make("'Pontiac'", test=False)

Documentation

cars.get_cars(test=True)

Returns the complete list of cars.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

cars.get_cars_by_year(year, test=True)

Returns all the cars for a given year.

  • year: int

    The year as an integer, between 1921 and 2013.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

cars.get_cars_by_make(make, test=True)

Returns all the cars of a certain make.

  • make: Str

    The make of the cars

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
