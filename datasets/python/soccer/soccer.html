Overview

Free open public domain soccer data (football). Leagues include the English Premier League, Deutsche Bundesliga, Spanish Primera Division ("La Liga"), Italian Serie A, and more.

https://github.com/openfootball/football.json

Usage

This library has 1 function you can use. 
import soccer
a_record = soccer.get_games("'Austrian Bundesliga'")

Documentation

soccer.get_games(league)

Returns information about all of the soccer games for a given league.

  • league: Str

    The league you want information about. Must be one of "Austrian Bundesliga", "Austrian Erste Liga", "English Premier League", "Spanish Primera Division", "Deutsche Bundesliga", "Deutsche Bundesliga 2", or "Italian Serie A".

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.
