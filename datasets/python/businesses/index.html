Overview

Yelp is providing all the data and reviews of the 250 closest businesses for 30 universities for students and academics to explore and research Yelp's dataset includes information for businesses near these 30 schools: Brown University, California Institute of Technology, California Polytechnic State University, Carnegie Mellon University, Columbia University, Cornell University, Georgia Institute of Technology, Harvard University, Harvey Mudd College, Massachusetts Institute of Technology, Princeton University, Purdue University, Rensselaer Polytechnic Institute, Rice University, Stanford University, University of California - Los Angeles, University of California - San Diego, University of California at Berkeley, University of Illinois - Urbana-Champaign, University of Maryland - College Park, University of Massachusetts - Amherst, University of Michigan - Ann Arbor, University of North Carolina - Chapel Hill, University of Pennsylvania, University of Southern California, University of Texas - Austin, University of Washington, University of Waterloo, University of Wisconsin - Madison, and Virginia Tech.

https://www.yelp.com/academic_dataset

Usage

This library has 5 functions you can use. 
import businesses
list_of_business = businesses.get_businesses_by_city("'blacksburg'")
list_of_business = businesses.get_business_by_name("'subway'")
list_of_business = businesses.get_all_businesses()
a_user = businesses.get_user_by_id("'UqNSytSc3hiYfKVCGwvLCg'")
list_of_user = businesses.get_users()
Additionally, some of the functions can optionally return Big Data using an extra argument. If you use this Big Data, it may be much slower. Wait until you are sure your code is correct before using these examples. 
import businesses
# These may be slow!
list_of_business = businesses.get_businesses_by_city("'blacksburg'", test=False)
list_of_business = businesses.get_business_by_name("'subway'", test=False)
list_of_business = businesses.get_all_businesses(test=False)
list_of_user = businesses.get_users(test=False)

Documentation

businesses.get_businesses_by_city(city, test=True)

Returns all the businesses in the given city.

  • city: Str

    The name of the city.

    If you make a typo, it will attempt to suggest a corrected answer. However, this is not perfect, so try to be as accurate as possible.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

businesses.get_business_by_name(name, test=True)

Returns all the businesses with the given name (matches fuzzily, so 'subway' would match 'Subway' and 'The subway').

  • name: Str

    The name of the business. It can be part of the name instead of the entire name.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

businesses.get_all_businesses(test=True)

Returns all the businesses in the database.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.

businesses.get_user_by_id(user_id)

Returns the information for a given user by their ID

  • user_id: Str

    A unique user id, given by the reviews of a business.

businesses.get_users(test=True)

Returns all of the users in the database.

  • optional
    test: boolean

    Optional argument to control whether ALL or SOME of the data is returned for this function. This makes it much easier to develop and test your code, only using a large (slow) amount of data when you are ready.
