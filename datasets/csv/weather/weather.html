Overview

Under the National Oceanic and Atmpospheric Administration, the National Weather Service provides daily weather reports for cities across the county. This is done through the use of 122 different Weather Forcast Offices throughout the country. These WFOs are responsible for the daily weather reports for serveral cities throughout their region of coverage. This data set takes the information from these WFO reports for cities across the country through the months of April to June of 2016.

http://w2.weather.gov/climate/

Field Descriptions

Key List of... Comment Example Value
Avg Temp Real number The average recorded temperature on this day, in degrees Farenheit. 34.0
Avg Wind Real number The average windspeed for that day, in Miles per Hour. 8.0
City String The city that the reporting station sends its data to. Note that the recording station itself might actually be in a different city. "Mount Holly/Philadelphia"
Code String The unique code representing this station. "ABE"
Day Integer number The day of the month that this report was made. 10
Full String A full string representation of the date this report was made. "04-10-2016"
High Gust Real number A gust is a sudden burst of wind (less than 20 seconds long), which is often much stronger than the average windspeed. This is reported in Miles per Hour. 31.0
High Wind Real number The highest windspeed for that day, in Miles per Hour. 22.0
Location String The exact location of this recording station. Note that the recording station itself might be in a different location than where it sends its data. "Mount Holly/Philadelphia, NJ"
Max Temp Real number The highest recorded temperature on this day, in degrees Farenheit. 40.0
Min Temp Real number The lowest recorded temperature on this day, in degrees Farenheit. 27.0
Month Integer number The month of the year that this report was made. 4
Precipitation Real number The average amount of rain, in inches. 0.19
State String The state that the reporting station sends its data to. Note that the recording station itself might actually be in a different state. "New Jersey"
Year Integer number The year that this report was made. 2016
