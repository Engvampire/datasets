Overview

Researchers have compiled a multi-decade database of the super-rich. Building off the Forbes World's Billionaires lists from 1996-2014, scholars at Peterson Institute for International Economics have added a couple dozen more variables about each billionaire - including whether they were self-made or inherited their wealth. (Roughly half of European billionaires and one-third of U.S. billionaires got a significant financial boost from family, the authors estimate.)

Field Descriptions

Key List of... Comment Example Value
age Integer number The current age of the billionaire. Ages that are represented as -1 stand for ages that were not available in the data that was collected. -1
category String A category representing where their money came from. "Financial"
citizenship String The name of the country that this billionaire has citizenship with. "Saudi Arabia"
company.name String The name of the company. "Rolaco Trading and Contracting Company"
company.type String The type of business for this company. "new"
country code String the 3-letter country code of the country where this billionaire has citizenship. "SAU"
founded Integer number The year that the company was founded. 1968
from emerging Integer number Whether the money came from emerging markets. True
gdp Real number The "Gross Domestic Product" of the country where the billionaire has citizenship. This is one of the primary indicators used to gauge the health of a country's economy. It represents the total dollar value of all goods and services produced over a specific time period; you can think of it as the size of the economy. 1.58e+11
gender String A string representing their gender. "male"
industry String The specific industry this billionaire profitted from. "Money Management"
inherited Integer number Whether the money was partially inherited. True
name String The name of the billionaire. "Abdul Aziz Al-Sulaiman"
rank Integer number The rank of this billionaire compared to the rest of the billionaires reported on. A lower rank means they make more money. 404
region String The region of the world where this billionaire lives. "Middle East/North Africa"
relationship String The billionaires relationship to the company. "founder"
sector String The sector of the business, or what segment of the economy they fit into. "construction"
was founder Integer number Whether the billionaire was the founder of their company. True
was political Integer number Whether the money came from politics. False
wealth.type String The type of billionaire that they are. "self-made finance"
worth in billions Real number The number of billion of dollars that this billionaire is worth. 1.0
year Integer number The year that data about this billionaire was collected. 1996
